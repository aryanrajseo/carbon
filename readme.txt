Carbon is a basic yet lightweight and powerful starter theme to design custom websites.

== Description ==
Use Carbon theme framework with our starter "Carbon Copy" or "Carbon Child" child theme.

== Frequently Asked Questions ==

Can I make changes to the carbon theme?
It is recommended to use child theme to make changes and prevent overwriting future updates.
